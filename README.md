# Зелёный агент или Green Agent

**Учебный проект по курсу "Управление IT-проектами"**

## О проекте

**Бизнес-модель:** B2C (Business-to-Consumer)

**Идея:**  
Мобильное приложение, которое связывает людей, накопивших вторсырьё (макулатуру, стекло, пластик, батарейки), с курьерами или волонтёрами, готовыми вывезти его в пункты приёма.  
Концепция: "Uber для сдачи вторсырья".

**Проблема:**  
Многие хотят сортировать отходы, но не имеют времени, машины или физической возможности доехать до пункта приёма.

**Решение:**  
Зелёный агент находит ближайшего проверенного курьера или волонтёра, который заберёт вторсырьё прямо из дома клиента за 30 минут. Пользователь получает чистую планету и бонусные баллы, а курьер – небольшой заработок.

**Целевая аудитория:**  
- Жители крупных городов с эко-привычками;
- Семьи с детьми (накапливается много картона, пластика);
- Люди без личного автомобиля.
